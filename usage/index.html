# Introduction

This package contains a collection of Laravel PHP classes designed to generate common HTML components. Initially it was developed as a extension of Laravel Collective but we decided to create a new component.

# Features

With Styde HTML you can built dynamically:

  • Fields such as: inputs, text, password, hidden, email, url, radio, select, file, textarea, etc.
  • Forms with a collections of fields and buttons.
  • Menus.
  • Alert messages.
  • Collection of radios and checkboxes.

Aditionally:

  • With internationalization.
